# Install a wallet
Learn how to get a wallet for storing and sending MINA

The Mina community has created a variety of different wallets.

You can send and receive MINA from all of these wallets, but currently only Auro Wallet supports zkApps interactions.

- <a href="https://www.aurowallet.com">Auro Wallet</a> - Chrome, App Store, Firefox, Google Play
- <a href="https://clor.io">Clorio Wallet</a> - Windows, OSX, Linux, and web browser
- <a href="https://www.staking-power.com">Staking Power</a> - Android and iOS
- <a href="https://www.ledger.com/mina-wallet">Ledger Hardware Wallet</a> - See Mina docs <a href="https://docs.minaprotocol.com/en/node-operators/ledger-app-mina">here</a>.

### Wallet for Node Operators

The Mina CLI Wallet is only intended for node operators. We do **not** recommend this Wallet for the average user.

View the docs for the Mina CLI <a href="https://docs.minaprotocol.com/en/node-operators/cli-reference">here</a>.
